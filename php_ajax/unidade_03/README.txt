204. Introdução ao JQuery

    JQuery

Definição: JQuery é uma biblioteca JavaScript.
Obs.: É a mais famosa entre os desenvolvedores.

Principais funções:
-> Acessar e manipular o DOM
-> Adicionar efeitos visuais e animações
-> Alterar o conteúdo de uma página
-> Alterar a apresentação de uma página
-> Buscar informações no servidor sem recarregar a página

Criada por: John Resig
"O foco principal da biblioteca JQuery é a simplicidade. Por que submeter os desenvolvedores ao martírio de escrever longos e complexos códigos para criar simples efeitos?" Livro JQuery in Action

Formas de usar:
-> Versão remota (CDN)
    Garante atualizações e correções de pequenos bugs automaticamente
    Pode ser mais lenta em alguns momentos
-> Baixar (depois publicar no provedor)
    Garante maior velocidade e confiabilidade
    Precisa atualizar manualmente
    Outras opções:
        -> Versão Production (32 Kb) - comprimida
            Recomendado para quem não tem experiência
        -> Versão Development (235 Kb) - não comprimida
            Recomendado para quem quiser abrir o código JQuery para fazer manipulações
    Link: https://jquery.com/
    Obs.: Versão 1.x do JQuery suporta navegadores mais antigos
