208. O que é um arquivo XML

    XML

Definição: Linguagem de marcação
Significado: Extensible Markup Language
XML se tornou padrão pela W3C em 1998

Utilização:
    Estruturar e descrever informações
    Planejado para ser usado pela internet

    Comunicação entre sistemas
        Envio de arquivos XML
        Serviços criados baseados em XML
